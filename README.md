# MERN Authentication App

A full-stack authentication application built with the MERN stack (MySQL, Express.js, React, Node.js) and containerized with Docker.

## Features

- User Registration & Login
- JWT Authentication & Refresh Tokens
- Protected Routes
- User Profile Management
- Password Change Functionality
- Responsive Design
- Docker Containerization
- Real-time State Management

## Tech Stack

**Frontend:**
- React 18
- TypeScript
- Tailwind CSS
- React Router DOM
- Vite

**Backend:**
- Node.js
- Express.js
- JWT Authentication
- MySQL
- bcryptjs

**DevOps:**
- Docker
- Docker Compose
